Promise

  • Promise is an object that returns a value in the future.
  • It represents the an eventual completion or failures of an asynchronous operation.
  • If a function cannot return a value or throw an exception without blocking, it can return a promise instead.
  • Use promise whenever we are using async or blocking code.
  • There are 3 PromiseStates: pending, fulfilled, and rejected
    • For multiple operations, after each promise has either resolved for rejected, we enter the pending state of the next promise in the chain.
  • resolve maps to then and reject maps to catch
    • .then() if resolved
    • .catch() if rejected
    • If something needs to be done in both cases use .finally

Promises vs Callbacks

  • You can achieve results from performing asynchronous operations using the callback approach or by using promises.
  • Unlike callbacks where you pass a callback into a function, with promise you attach callbacks on the returned promise object.
  • The common issue for using the callback approach is that when we perform multiple asynchronous operations simultaneously, we can easily end up with callback hell. With promises, we can use .then() method to chain the codes at the root of the first handler.
//Callbacks:
function getMoneyBack(money, callback) {
  if (typeof money !== 'number') {
    callback(null, new Error('money is not a number'));
  } else {
    callback(money);
  }
}

const money = getMoneyBack(1200);
console.log(money);

// Promises:
function getMoneyBack(money) {
  return new Promise((resolve, reject) => {
    if (typeof money !== 'number') {
      reject(new Error('money is not a number'))
    } else {
      resolve(money);
    }
  });
}

getMoneyBack(1200).then(money) => {
  console.log(money);
}

Creating Promise

new Promise(function(resolve, reject) => { ... });

const myPromise = new Promise((resolve, reject) {
  let connection = true;

  if (connection) {
    resolve('Connection established');
  } else {
    reject('Connection refused');
  }
});

// if resolved
myPromise.then((message) => {
  console.log(message);
// if rejected
}).catch((message) => {
  console.log(message);
});

Promise Chaining

  • Useful when we need to execute a chain of asynchronous tasks.
  • Each task that’s being chained can only start as soon as the previous task has completed, controlled by the .then() method of the chain.
  • Each promise represents the completion of another asynchronous step in the chain.

Promise Prototype Methods

  1. Promise.prototype.then(onFullfilled, onRejected): to schedule a callback to be executed when the promise is successfully resolved
  2. Promise.prototype.catch(onRejected): if you want to schedule a callback to be executed when the promise is rejected
  3. Promise.prototype.finally(onFinally): to executed the same piece of code whether the promise is fulfilled or rejected.

Promise Helper Functions

  1. Promise.all(iterable): to accumulate a batch of asynchronous operations and receive each of their values as an array; promise rejects with an error if at least one of the operations failed; used if multiple asyncrhonous tasks are dependent on each other.
  2. Promise.race(iterable): returns a promise that either fulfills or rejects whenever one of the promises in an iterable resolves or rejects with either the value or the reason from that promise.
  3. Promise.allSettled(iterable): returns a promise that resolves after all of the given promises have either fulfilled or rejected, with an array of objects that each describes the outcome of each promise; used when multiple asynchronous tasks are not dependent on one another to complete successfully.
  4. Promise.any(iterable): takes an iterable of Promise objects and returns a single promise that resolves as soon as any of the promises in the iterable fulfills, with the value of the fulfilled promise.

Promise to async/await

await: wait for the promise to resolve; only works for asynchronous functions

Just using promises and its methods:

fetch(wordnikAPI) // fetch returns a promise
  .then(response => response.json()) // .json returns a promise
  .then(json => {
    createP(json.word);
    return fetch(giphyAPI + json.word); // return to chain promises!
  })
  .then(response => response.json())
  .then(json => {
    createImg(json.data[0].images['fixed_height_small'].url);
  })
  .catch(err => console.log(err));

Converting the code above using async and await:

function setup() {
  noCanvas();
  wordGIF().
    then(results => {
      createP(results.word);
      createImg(results.img);
    }).
    catch(err => console.error(err));
}

async function wordGIF() {
  let response1 = await fetch(wordnikAPI);
  let json1 = await response1.json();
  let response2 = await fetch(giphyAPI + json1.word);
  let json2 = await response2.json();
  let img_url = json2.data[0].images['fixed_height_small'].url;

  return {
    word: json1.word,
    img: img_url
  }
}
